# 🚐 Live Van Tracking System

## 📌 Project Overview
The Live Van Tracking System is a real-time vehicle tracking solution that allows users to monitor the exact location of vans using GPS technology.It Provide the free of cost beacuse we do not use the any hardware device into this project.

## 🎯 Features
- 📍 Real-time vehicle tracking
- 🗺️ Map-based interface
- 👤 User, Driver & Admin modules
- ⏱️ Estimated arrival time
- 🔐 Secure login system
